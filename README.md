KUTUN TERMINAL | Geopolitical Macro & Equity Intelligence Terminal
A high-performance financial intelligence platform designed to quantify capital market cash flows and asset volatility following geopolitical conflicts and macro shocks.


Main Intelligence Hub

The Main Intelligence Hub provides a unified view of global news, sector capital flows, and real-time commodity barometers.

Historical Impact Engine

When a user selects a macro event, like the 2011 US Credit Downgrade, the terminal instantly snaps all cross-asset charts to that precise timeframe. Crucially, it re-calculates the relative 1-day, 1-week, and 1-month sector cash flows for that specific shock and automatically displays the Highest Earner and Loser assets. This feature turns a chaotic event into a tactical blueprint.

Integrated Equity Terminal (US & TSX)

To mimic an institutional workflow, I built a custom Equity Terminal that provides professional-grade customization. It utilizes a decoupled Python data pipeline to serve fundamental data, allowing users to instantly sift through US (S&P 100) or CAN (TSX) market sectors, providing crucial localized context for Canadian investors like Wealthsimple users.

FX & Commodities Cross-Correlator

This tool allows for complex cross-asset analysis. It is designed to be dual-purpose: users can utilize it to overlay safe-haven commodities onto the specific historical date range of a macro event selected from the main terminal, or use it as a standalone, zero-latency commodities comparison tool to track long-term performance shifts.
